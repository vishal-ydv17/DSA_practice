## Quick Overview
- **Problem Type**: Array, Two Pointers  
- **Difficulty**: Easy  
- **Key Concept**: Find increasing then decreasing sequence  
- **Solution Summary**:  
  Traverse the array while strictly increasing, then strictly decreasing.  
  A valid mountain must:
  - Increase for at least one element (peak not at start),
  - Decrease for at least one element (peak not at end),
  - Have exactly one peak.
