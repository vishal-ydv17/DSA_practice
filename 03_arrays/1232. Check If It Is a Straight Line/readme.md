- **Problem Type**: Geometry  
- **Difficulty**: Easy  
- **Key Concept**: Collinearity using slope cross-multiplication  
- **Solution Summary**:  
  All points lie on a straight line if the **slope between every pair of adjacent points is the same**.  
  Use cross-multiplication to avoid division, checking:  
  \[
  (y2 - y1)(x3 - x2) = (y3 - y2)(x2 - x1)
  \]
