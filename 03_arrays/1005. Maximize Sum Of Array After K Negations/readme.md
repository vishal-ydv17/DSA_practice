## Quick Overview
- **Problem Type**: Array, Greedy, Sorting  
- **Difficulty**: Easy  
- **Key Concept**: Maximize sum by negating smallest values  
- **Solution Summary**:  
  1. Sort the array in ascending order.  
  2. Negate the smallest negative numbers **first** while `k > 0`.  
  3. If `k` remains odd after all negatives are used, flip the smallest absolute value.  
  4. Compute the final sum.
