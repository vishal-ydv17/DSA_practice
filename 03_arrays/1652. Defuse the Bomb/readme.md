## Quick Overview
- **Problem Type**: Array, Simulation  
- **Difficulty**: Easy  
- **Key Concept**: Circular neighbor sum  
- **Solution Summary**:  
  For each index `i`:
  - If `k > 0`: replace `nums[i]` with the sum of the next `k` numbers (circularly).
  - If `k < 0`: replace `nums[i]` with the sum of the previous `|k|` numbers (circularly).
  - If `k == 0`: result at index `i` becomes `0`.
